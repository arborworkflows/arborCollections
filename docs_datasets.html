Arbor datasets

  • anolis.phy: a phylogenetic tree of anoles
  • anolis.csv: character data for anoles (comma-separated format)
  • phelsuma.phy: a phylogenetic tree of day geckos, from this paper.
  • phelsuma.csv: character data for day geckos (comma-separated format), from this paper.
