Arbor Functions and Workflows

Arbor functions and workflows are divided into various collections. You can access help for functions in arbor collections here.

  • base: Basic functions to load and manipulate data.
  • phylogeny: Plot and analyze phylogenetic trees.
  • datasets: Example datafiles for Arbor.
